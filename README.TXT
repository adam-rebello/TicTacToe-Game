PROJECT TITLE: Tic Tac Toe Game
PURPOSE OF PROJECT: To play a game of TicTacToe against a friend.
VERSION or DATE: April 9th, 2024
HOW TO START THIS PROJECT: Create a new object from the TicTacToeGUI Class and name it anything you would like, then click okay and the GUI will appear.
AUTHORS: Adam Rebello 
USER INSTRUCTIONS: Decide who is player x and player o. PLayer x starts and clicks on a square then player o does their turn. After playing you will see if there is a winner or if it tied. Then you can click new game or quit if you do not want to play anymore.
